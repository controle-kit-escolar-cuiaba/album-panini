## ⚡ COMEÇAR EM 5 MINUTOS

### **Passo 1: Preparar o Código**

Abra o arquivo `index.html` em um editor de texto e:

1. Procure pela linha com `[TODO: Cole aqui...]`
2. Copie **TODO** o código do arquivo `figurinhas-panini.jsx`
3. Cole no lugar do `[TODO...]`
4. Remova a primeira linha `import React...` se tiver
5. **Salve o arquivo**

### **Passo 2: Hospedar Online** (RECOMENDADO)

**Opção A - Vercel (Mais rápido):**
1. Vá para vercel.com
2. Crie conta (ou use GitHub)
3. Clique "Add new site"
4. Faça upload dos 3 arquivos (`index.html`, `manifest.json`, `sw.js`)
5. Deploy automático em segundos!

**Opção B - GitHub Pages:**
1. Crie repositório no GitHub
2. Upload dos arquivos
3. Settings → Pages → Main branch
4. Pronto! Link gerado automaticamente

**Opção C - Netlify:**
1. netlify.com → "Add new site"
2. Arraste os arquivos
3. Deploy instantâneo!

### **Passo 3: Instalar no Celular**

**Android:**
1. Acesse o link no Chrome
2. Menu (⋮) → "Instalar app"
3. Confirma
4. Ícone aparece! 📱

**iPhone:**
1. Acesse o link no Safari
2. Compartilhar (↑) → "Adicionar à Tela de Início"
3. Confirma
4. Ícone aparece! 📱
